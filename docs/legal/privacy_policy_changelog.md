## Material Changes Definition

Material changes require user notification (30 days' notice):

- Changes to data collection practices
- Changes to data sharing or disclosure
- Changes to data subject rights procedures
- Changes to international data transfers
- Changes to data retention policies
- Changes to security measures (if reduced)

Non-material changes (notification not required):

- Clarifications or formatting improvements
- Updates to contact information
- Corrections of typos or errors

---

## Compliance Notes

- **GDPR:** Material changes require notification and may require re-consent
- **CCPA/CPRA:** Material changes require notification (30 days' notice)
- **Documentation:** All changes must be documented in this changelog
- **Version Control:** Track changes in git (markdown source)
